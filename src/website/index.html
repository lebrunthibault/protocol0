Background story

Why Protocol 0 exists

Hey, my name is Thibault and I work in software as well as produce music. I love Ableton even though it has a few rough edges, like its limited keymapping abilities.

I've been working around this by using custom remote scripts and shortcuts for a while and wanted to share this project with you guys. I hope you find it helpful.
