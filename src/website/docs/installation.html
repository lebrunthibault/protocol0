Getting started

Installation

Requirements

  • Ableton Live 12. The installer auto-detects your install.
  • Windows. macOS coming soon.

Install

  1. Download and run the latest installer.
  2. Protocol 0 launches and you can start configuring your shortcuts.
  3. Enable the Protocol 0 control surface in Live. (Preferences → Tempo & Midi → Control Surface → Protocol 0)

SmartScreen warning. Releases are now code-signed (Azure Trusted Signing), so the installer shows a verified publisher in the UAC prompt. SmartScreen reputation still builds up over the first downloads, so it may warn for a while on a fresh release — click More info → Run anyway.

Updating

Just download and install a new version. Your config is kept (in %APPDATA%\Protocol0\shortcuts.json).

Uninstalling

Use Add or Remove Programs in Windows. The uninstaller removes everything including your shortcut config.

Want to run from source? See the README for the developer setup.
